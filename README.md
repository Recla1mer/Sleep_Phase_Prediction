# Neural Networks for Sleep Analysis

Build Convolutional Neural Networks for Sleep Stage Classification and Sleep Apnea Detection based on heart and wrist-movement activity.
The main.py file should offer enough comments to get you started.

Just a brief note: I haven’t included my results here because I was unsure about the data privacy implications. However, you can obtain them from my supervisor if you would like to use the trained networks. Otherwise, you would need to train the models on your own data.
