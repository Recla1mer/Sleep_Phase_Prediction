# Neural Networks for Sleep Analysis

Build Convolutional Neural Networks for Sleep Stage Classification and Sleep Apnea Detection based on heart and wrist-movement activity.
Would usually add a proper description, but who is gonna end up here anyway?
But do not worry, the main.py file should offer enough comments to get you started.

Just a brief note: I haven’t included my results here because I was unsure about the data privacy implications. However, you can obtain them from my supervisor if you would like to use the trained networks. Otherwise, you would need to train the models on your own data.

Please mind that at the end of my master's thesis, when I needed results preferably yesterday, I just solved problems as fast as possible and started getting a bit sloppy with the comments, so for a minor set of functions they might be outdated. I know, I despise myself for this too, but don't worry: for the major part the code should be clean and well documented.
